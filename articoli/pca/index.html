Analisi delle Componenti Principali (PCA)

L’Analisi delle Componenti Principali (PCA – Principal Component Analysis) è una tecnica statistica utilizzata per ridurre la dimensionalità dei dati mantenendo però la maggior parte dell’informazione originale.
Il suo scopo è trasformare un insieme di variabili spesso correlate in un numero minore di variabili non correlate, chiamate componenti principali.

In altre parole, la PCA comprime i dati senza perdere (quasi) nulla della loro struttura informativa.

Concetto intuitivo

Immagina una nuvola di punti in uno spazio tridimensionale.
La PCA cerca:

  • la direzione lungo cui i dati sono più dispersi: questa è la prima componente principale (PC1)
  • una seconda direzione perpendicolare alla prima che cattura la massima variabilità residua: PC2
  • e così via

Ridurre le dimensioni significa proiettare i dati su questi nuovi assi e scartare quelli che spiegano poca varianza, spesso associati a rumore o ridondanza.

I passaggi matematici della PCA

1. Standardizzazione

Le variabili vengono portate sulla stessa scala (media = 0, varianza = 1).
Questo evita che variabili con valori numericamente più grandi dominino l’analisi.

2. Matrice di covarianza

Si calcola come le variabili variano insieme.
La covarianza misura la relazione lineare tra coppie di variabili.

3. Autovalori e autovettori

  • Gli autovettori rappresentano le nuove direzioni (componenti principali).
  • Gli autovalori indicano quanta varianza spiega ciascuna componente.

4. Scelta delle componenti

Si ordinano gli autovalori dal più grande al più piccolo e si selezionano le prime (k) componenti, ad esempio quelle che spiegano il 95% della varianza totale.

5. Proiezione

I dati originali vengono trasformati nel nuovo spazio a (k) dimensioni.

Quando si usa la PCA?

  • Visualizzazione: ridurre da 10–20 dimensioni a 2–3 per creare grafici interpretabili.
  • Velocizzare algoritmi di Machine Learning: meno variabili → meno calcoli.
  • Rimozione del rumore: le componenti con varianza minima spesso rappresentano disturbi.
  • Gestione della multicollinearità: quando molte variabili sono fortemente correlate.

Un esempio pratico

Supponi di avere un dataset con decine di misurazioni fisiche di un atleta: battito cardiaco, ossigenazione, velocità, forza, resistenza, ecc.
La PCA potrebbe rivelare che bastano due sole componenti per descrivere:

  • la sua forma fisica generale
  • la sua capacità di recupero

Riducendo drasticamente la complessità dell’analisi senza perdere informazione rilevante.

Nota importante

La PCA è un metodo lineare.
Se i dati presentano relazioni non lineari, possono essere più adatti metodi come:

  • t-SNE
  • UMAP
  • Kernel PCA

import org.ejml.simple.SimpleMatrix;

public class SimplePCA {

    public static void main(String[] args) {

        // Matrice dei dati (ogni riga = osservazione, ogni colonna = variabile)
        double[][] rawData = {
                {2.5, 2.4},
                {0.5, 0.7},
                {2.2, 2.9},
                {1.9, 2.2},
                {3.1, 3.0},
                {2.3, 2.7},
                {2.0, 1.6},
                {1.0, 1.1},
                {1.5, 1.6},
                {1.1, 0.9}
        };

        SimpleMatrix data = new SimpleMatrix(rawData);

        // 1. Standardizzazione
        SimpleMatrix standardized = standardize(data);

        // 2. Matrice di covarianza
        SimpleMatrix cov = covarianceMatrix(standardized);

        // 3. Autovalori e autovettori
        var eig = cov.eig();
        SimpleMatrix eigenVectors = eig.getV();

        // 4. Proiezione sui primi 2 autovettori
        SimpleMatrix projected = standardized.mult(eigenVectors);

        System.out.println("Dati proiettati sulle componenti principali:");
        projected.print();
    }

    private static SimpleMatrix standardize(SimpleMatrix data) {
        int rows = data.numRows();
        int cols = data.numCols();
        SimpleMatrix result = new SimpleMatrix(rows, cols);

        for (int j = 0; j < cols; j++) {
            double mean = 0;
            for (int i = 0; i < rows; i++) mean += data.get(i, j);
            mean /= rows;

            double variance = 0;
            for (int i = 0; i < rows; i++)
                variance += Math.pow(data.get(i, j) - mean, 2);
            variance /= rows;

            double std = Math.sqrt(variance);

            for (int i = 0; i < rows; i++)
                result.set(i, j, (data.get(i, j) - mean) / std);
        }

        return result;
    }

    private static SimpleMatrix covarianceMatrix(SimpleMatrix data) {
        int n = data.numRows();
        SimpleMatrix transposed = data.transpose();
        return transposed.mult(data).divide(n - 1);
    }
}
